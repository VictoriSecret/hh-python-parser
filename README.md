# Парсер вакансий Python с hh.ru

Простой парсер для сбора вакансий по ключевому слову "Python" с сайта hh.ru.

## Возможности

- Поиск вакансий по ключевому слову
- Парсинг названия, компании, зарплаты, ссылки, опыта и города
- Сохранение результатов в CSV файл
- Настройка количества страниц

## Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone https://github.com/VictoriSecret/hh-python-parser.git
cd hh-python-parser
```
2. Установите зависимости:

```bash
pip install requests beautifulsoup4 pandas
```
3. Запустите парсер:
```bash
python hh_parser.py
```
